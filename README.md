## Pokemon List Application

### Stack
- Node 20.14.0
- npm 10.9.0

### Commands
1. Get project
```bash
  git clone https://github.com/MaxainNN/PokemonListReact.git
```
2. Install dependencies
```bash
  npm install
```

3. Run app
```bash
  npm run dev
```

4. Open http://localhost:5173/
